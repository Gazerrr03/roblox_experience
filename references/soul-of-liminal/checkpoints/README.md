This folder tracks design-conversation intervals.

Recommended flow:

1. User says `checkpoint`.
2. Open a new interval file if none is open.
3. Capture fragments during the conversation in candidate docs or the open interval file.
4. When the user says `checkpoint` again, distill the interval into work items and archive it.

Use open intervals for raw material.
Use closed intervals for structured work items.
