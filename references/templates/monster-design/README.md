# 怪物策划模板

这个文件夹用于“先策划、后统一执行”的怪物设计草案。

## 使用方式

1. 在 `Monster-Current-Baseline.csv` 维护当前已上线怪物的执行基线。
2. 在 `Monster-Design-Gaps.csv` 记录当前缺口与待补策划项。
3. 在 `Monster-Next-Batch-Plan.csv` 为下一批怪物预填参数与定位。
4. 新怪物请使用唯一怪物 ID（例如 `M002`、`M003`）并按行追加。
5. 你全部设计完成后，我们再统一实现。

## 推荐命名

- `scrap-wisp`（当前基线怪物）
- `M002`（下一批候选）
- `M003`（下一批候选）

## 统一执行前检查清单

- 每个怪物都有完整的移动/生命/攻击参数。
- 每个怪物都有明确的“玩家反制窗口”。
- 每个怪物都有可验证的验收标准。
- 任何跨系统依赖（地图、状态、共享规则）已写清楚。
