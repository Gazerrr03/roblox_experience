# 怪物策划模板

这个文件夹用于“先策划、后统一执行”的怪物设计草案。

## 使用方式

1. 复制 `Monster-Design-Template.md`。
2. 重命名为 `MXXX-怪物名.md`。
3. 按模板填完整个战斗链路和关键数值。
4. 每个怪物单独一份。
5. 你全部设计完成后，我们再统一实现。

## 推荐命名

- `M001-ScrapWisp.md`
- `M002-<怪物名>.md`
- `M003-<怪物名>.md`

## 统一执行前检查清单

- 每个怪物都有完整的移动/生命/攻击参数。
- 每个怪物都有明确的“玩家反制窗口”。
- 每个怪物都有可验证的验收标准。
- 任何跨系统依赖（地图、状态、共享规则）已写清楚。
