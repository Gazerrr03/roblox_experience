## Feature Manifest

`places/run/FEATURES.md` 是 run place 的功能权威清单。规则见 root `AGENTS.md`：

- 每条 PR 必须同步更新 `FEATURES.md`，否则视为未完成
- 功能状态：`active` = 已实现，`⚠️ missing` = 已知缺失
- 章节固定，不可删减（见 root AGENTS.md Feature Manifest 一节）
- Studio 创作需求记录在第 9 章 Authoring Requirements
